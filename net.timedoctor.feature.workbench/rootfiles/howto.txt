To build TimeDoctor for your Linux distro/version/platform, please follow these 
steps:

Downloads:
1) If not present, please download Eclipse for RCP/RAP developers from 
   http://download.eclipse.org/eclipse/downloads/
   for your relevant Linux distro/version/platform
2) If not present, please download a Java jre: http://wiki.eclipse.org/Eclipse/Installation
   

Setup:
1) Start Eclipse
2) Import TimeDoctor projects from the TimeDoctor GIT repository. For information on TimeDoctor 
   GIT access, please see http://sourceforge.net/git/?group_id=174793. For
   information on using GIT with Eclipse, please see the eGit plugin documentation:
   http://wiki.eclipse.org/EGit/User_Guide

RCP export:
1) Double-click the workbench.product file in the project
   net.timedoctor.feature.workbench
2) In the Overview sheet, click the "Eclipse Product export wizard" link and follow
   the online instructions
3) To export for different platforms, install the RCP delta pack: http://ugosan.org/eclipse-rcp-delta-pack/

If you face problems, please raise a support request at     
http://sourceforge.net/tracker/?func=add&group_id=174793&atid=870647

Good Luck!
